The following vocabulary list will help enhance your English level. I've compiled this myself and I'm sharing it with you.

  • a lot of → a tremendous amount of
  • admit → concede
  • after → following
  • although → albeit
  • always → invariably
  • among → amid
  • angry → furious/outraged/blow a fuse
  • annoying → irritating
  • approve of → applaud
  • ask → inquire
  • ask for → request
  • attack → assault
  • bad → awful/horrendous/atrocious
  • beautiful → gorgeous/elegant/stunning/engaging/aesthetic/breathtaking
  • before → preceding
  • begin → initiate
  • big → large/huge/enormous/gigantic/massive/tremendous/immense
  • busy → swamped
  • buy → purchase
  • calm → peaceful
  • cause → trigger/induce/arouse
  • change → modify/amend/alter
  • clean → sanitize
  • complete → utter
  • cute → adorable
  • dangerous → hazardous
  • destroy → eradicate
  • difficult → challenging
  • disappointed → disillusioned/disheartened
  • disgusting → revolting
  • early → premature
  • easy → simple/unchallenging/effortless/uncomplicated/a piece of cake
  • everywhere → ubiquitous
  • expect → anticipate
  • expensive → extravagant
  • famous → prominent/renowned/notable
  • fast (adj.) → speedy/rapid/swift
  • fast (adv.) → rapidly/swiftly
  • fiind → discover
  • friend → companion
  • funny → hilarious
  • give → provide/allocate
  • go → hit the road
  • good → magnificent
  • happy → delighted/elated/over the moon
  • hard → difficult/challenging/demanding/arduous/strenuous
  • hardworking → diligent/industrious/studious/laborious
  • have (v.) (of a place) → boast
  • help → assist
  • help out → come to the rescue
  • hide → conceal
  • honest → frank
  • hope → desire/aspiration
  • hungry → starving
  • important → crucial/vital//critical/essential/significant/imperative/paramount
  • interesting → intriguing/fascinating
  • keep → preserve
  • kind → affectionate/benevolent/compassionate/gracious/philanthropic
  • large → immense
  • later → subsequently
  • learn → acquire
  • look back → reflect
  • lose → misplace
  • lucky → fortunate
  • many → innumerable
  • necessary - > indispensable
  • nervous -> anxious/tense/jittery/butterflies in the stomach
  • next to - > adjacent to
  • nice - > pleasant

  • pick -> select
  • polite -> courteous

  • similar to - > analogous to/akin to
  • small - > miniature/slight
  • smart -> brainy/clever/intelligent/acute/savvy/quick-witted
  • soon - > imminently
  • sorry - > apologetic
  • start - > commence
  • strong - > robust
  • stupid -> dumb/foolish/unintelligent/unwise/dense/idiotic/slow-witted
  • suddenly - > abruptly
